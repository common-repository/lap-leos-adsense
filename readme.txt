This plugin allows add AdSense code to your posts with different ID. Useful, if you have authors of your pages with their own AdSense settings.

== Description ==

This plugin allows add AdSense code to your posts. Important is, that this plugin allows you to specify different ID and channel ID or ad-slot ID (Useful, if you have several authors of your pages with their own AdSense settings) for each post. All IDs are specified at each post separately. Plugin can identify how old is your post also and can show AdSense ads for posts older than days you specify.

== v0.5 - first release ==

Version 0.5 supports:
(Developed and tested on WP version 2.5 - 2.5.1)
* Specification of pub-ID at each post.
* Specification of ad-slot/channel-ID at each post.
* Specification for how min. old posts (in days) should plugin handle ads.

== Installation ==

1. Upload `leos-adsense-plugin.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Insert code to be inserted by the plugin in 'Settings->LAP' menu.
1. Specify in days for how old the post must be to use with this plugin in 'Settings->LAP' menu.
1. Specify pub-ID and (if you want)  ad-slot/channel-ID at each post at each post separately.
1. Insert '&lt;!--[LAP-plugin1]--&gt;', &lt;!--[LAP-plugin2]--&gt;, .... &lt;!--[LAP-plugin5]--&gt; tags anywhere into your posts.
1. [Complete usage instructions are available here.](http://www.radime.cz/wordpress/plugins/lap-leos-adsense/)
